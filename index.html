My Projects

Pokemon Evolutionary Algorithm

An exploration to what extent an evolutionary algorithm can determine the optimal traits of a pokemon team. Programmed in Java. I also wrote a paper that discussed the results of the algorithm and overall appraoch to the project.

Pokemon Simulator

A text-based Pokemon Battle Simulator. Includes support up to the 5th Generation of the Pokemon. Written in Java, this simulator applies inheritance to quickly establish the complex relationships between different mechanics found within the Pokemon games. Hash maps were utilized to create a large Pokemon library that provides players many options on each game.

Web Enabled Security System

Using a Rasberry Pi 2, PIR motion sensors, and a TTL serial camera I developed a motion detector security camera. When motion was detected with the motion sensors, the TTL camera will take a picture and send an email to me. The camera was controlled using the picamera library and the GPIO library was used to receive input from the PIR motion sensor.
